The imported dataset is in a wide format, for data analysis it will be converted to a long format with only three columns; “country”, “year” and “mortality”. Child mortality is for 0 to 5 year olds dying per 1000 born.

child_mortality_long = child_mortality.melt(id_vars='country', var_name= 'year', value_name='mortality')
